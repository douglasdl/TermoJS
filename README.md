[Rapid API](https://rapidapi.com/sheharyar566/api/random-words5/) - Random Words API.

## Create the backend project
```bash
npm init -y
```

## Install the dependencies
```bash
npm i axios cors express nodemon dotenv
```

## Run the backend
```bash
npm run start:backend
```
